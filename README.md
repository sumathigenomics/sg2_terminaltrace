# sg2_terminaltrace
is designed as a high-speed, lightweight Python engine for 5' and 3' end-motif analysis of cfDNA and sequencing data. Developed by Dr. Saminathan Sivaprakasham Murugesan.

SG2: TerminalTrace - A high-performance terminal motif analyzer for cfDNA and NGS data. Developed by Sumathi Genomics. Fast, memory-efficient k-mer extraction from FASTQ/GZ
